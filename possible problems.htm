Possible Problems

Q: Neural network can't make phrase with my word
A: It's only a beta-version, it has limited number of words. In final version this problem will be fixed. All words that can be used are in their wordlists. Check txt files

Q: Neural network turns off when I'm entering my word
A: Make shure that all files are in their appropriat directories. It might be missing a wordlist

Q: Neural Network give me one illogical phrase many times. It can't remember anything
A: Check if txt file called "exceptions" is on it's place. It have to be in the same directory as all other txts
